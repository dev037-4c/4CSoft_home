4CSoft – LX2 학습관리시스템 홈페이지

토스 페이먼츠 스타일의 깔끔하고 모던한 UI/UX를 기반으로 제작된 LX2 학습관리시스템 소개 페이지입니다.
부드러운 인터랙션, 반응형 레이아웃, 신뢰감 있는 컬러 톤을 통해 사용자에게 직관적이고 편안한 경험을 제공합니다.

🎨 디자인 특징

미니멀 & 클린 UI — 불필요한 장식 없이 콘텐츠 중심의 레이아웃

부드러운 인터랙션 — 페이드인, 스크롤, 호버 등 자연스러운 애니메이션

완전 반응형 디자인 — 모바일·태블릿·PC에서 최적화된 화면 제공

신뢰감 있는 블루 톤 컬러 — 금융 서비스 스타일의 안정적 분위기

넓은 여백과 가독성 높은 구성 — 정보 전달에 집중한 콘텐츠 설계

📁 프로젝트 구조
4csoft_homepage/
├── index.html          # 메인 페이지
├── css/
│   └── style.css       # 전체 스타일
├── js/
│   └── main.js         # 인터랙션 및 애니메이션 스크립트
├── assets/             # 이미지 및 정적 리소스
└── README.md           # 프로젝트 문서

🚀 시작하기
1. 클론
git clone [repository-url]
cd 4csoft_homepage

2. 로컬 서버 실행
# Python
python -m http.server 8000

# Node.js
npx serve

# 또는 VSCode Live Server


브라우저에서 http://localhost:8000 으로 접속합니다.
